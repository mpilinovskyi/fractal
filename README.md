# Фрактальний Експлорер Серпінського

Цей веб-додаток є інтерактивним інструментом для візуалізації та дослідження фракталів. Він демонструє різні підходи до створення фрактальних структур: від класичної рекурсії до систем ітерованих функцій (IFS) та L-систем.

## Можливості
* **Трикутник Серпінського:** Рекурсивна побудова з інтерактивним масштабуванням та динамічним рівнем деталізації (LOD).
* **Килим Серпінського:** Побудова за допомогою методу випадкових ітерацій (Chaos Game) з підтримкою зуму.
* **Крива Серпінського:** Візуалізація через L-системи (системи Лінденмаєра) з підтримкою "черепашачої графіки".
* **Інтерактивність:** Можливість панорамування (drag & drop) та масштабування (zoom) за допомогою миші для всіх видів фракталів.
* **Оптимізація:** Використання технік відсікання за областю видимості (culling) для забезпечення плавної роботи навіть при глибокому наближенні.

## Як почати
Проєкт не потребує встановлення чи збірки. Оскільки це чистий HTML/JS, ви можете просто відкрити файл `triangle.html` у будь-якому сучасному веббраузері.
